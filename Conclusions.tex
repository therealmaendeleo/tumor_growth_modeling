\chapter*{Заключение}
\addcontentsline{toc}{chapter}{Заключение}

В ходе выполнения настоящей научно-исследовательской работы была успешно решена задача
разработки и верификации программно-модельного комплекса для анализа динамики опухоли при
иммунотерапии. Актуальность работы продиктована необходимостью в инструментах для
прогнозирования и оптимизации современных методов лечения рака.

За основу была взята математическая модель на базе обыкновенных дифференциальных
уравнений, описывающая взаимодействия между опухолевыми и иммунными клетками. Модель была
реализована в виде интерактивного приложения на языке Python, которое стало гибкой
платформой для проведения вычислительных экспериментов.

Проведенный параметрический анализ выявил ключевые свойства системы, в частности, наличие
критических, пороговых значений для параметров иммунного ответа, при переходе через
которые исход заболевания кардинально меняется. Моделирование различных терапевтических
стратегий продемонстрировало предсказательную силу модели: она показала, что монотерапия
может быть недостаточной, в то время как комбинированные подходы способны достигать
синергетического эффекта, приводя к полному уничтожению опухоли.

Ключевым итогом работы стала успешная валидация модели путем сравнения ее результатов с
опубликованными экспериментальными данными, где было достигнуто хорошее соответствие
(коэффициент детерминации $R^2 = 0.74$). Это подтвердило адекватность разработанного
подхода и корректность заложенных в него механизмов.

Таким образом, цель работы была полностью достигнута. Созданный и проверенный
инструментарий представляет ценность для дальнейшего изучения иммунотерапевтических
протоколов и может быть усовершенствован в будущем, например, путем добавления в модель
пространственных или стохастических эффектов.

В процессе выполнения работы были освоены компетенции, предусмотренные учебным планом:

УК-1: Способен осуществлять поиск, критический анализ и синтез
информации, применять системный подход для решения поставленных задач.
В рамках работы проводился систематический анализ научной литературы для
формирования математической модели и
поиска данных для ее валидации. Системный подход был применен при декомпозиции
задачи на подзадачи: математическое описание, программная реализация,
проведение вычислительных экспериментов и верификация.

УК-4: Способен осуществлять деловую коммуникацию в устной и письменной
формах на государственном языке Российской Федерации и иностранном(ых) языке(ах).
Устная коммуникация осуществлялась в рамках взаимодействия с научным
руководителем. Письменная коммуникация реализована в процессе подготовки
данного отчета на государственном языке. В работе также использовались
англоязычные научные источники и техническая документация.

ПК-1: Способен проводить научно-исследовательские и опытно-конструкторские
разработки.
В рамках работы была проведена опытно-конструкторская разработка программного
продукта на языке Python с использованием библиотеки PyQt5. Разработанное
приложение с графическим интерфейсом позволяет в интерактивном режиме задавать
параметры модели и начальные условия, выбирать различные сценарии терапии
(постоянная, импульсная) и немедленно визуализировать результат в виде графиков.

ПК-2: Способен использовать методы компьютерного моделирования,
современные методы обработки информации и оформлять полученные результаты в
виде отчетов, презентаций и докладов.
В работе были применены методы компьютерного моделирования на основе системы
ОДУ для описания биологического процесса. Для решения системы использовался
численный метод Рунге-Кутты 4-го порядка. Были использованы современные методы
обработки информации, в частности, автоматизированный подбор параметров модели
с помощью методов оптимизации из библиотеки SciPy для минимизации расхождения с
экспериментальными данными. Все полученные результаты были проанализированы и
оформлены в виде текста, таблиц и графиков в данной работе.
