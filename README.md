# Компьютерное моделирование динамики опухолевых процессов при иммунотерапии

Репозиторий для научно-исследовательской работы (НИР) по теме "Компьютерное моделирование динамики опухолевых процессов при иммунотерапии". Проект включает в себя файлы отчета на базе издательской системы LaTeX и программную реализацию модели.

## Как собрать PDF

Для компиляции LaTeX-документа и получения `Thesis.pdf` потребуется компилятор LaTeX (я использовал `pdflatex`) и `biber` для обработки библиографии.

**Рецепт** для получения pdf-файла из исходников (рекомендуется настроить через расширение [LaTeX Workshop](https://github.com/James-Yu/LaTeX-Workshop) в редакторе VS Code):

```bash
pdflatex Thesis.tex
biber Thesis
pdflatex Thesis.tex
pdflatex Thesis.tex
```

## Структура проекта

- `Thesis.tex`: Главный файл LaTeX, объединяющий все части документа.
- `Title.tex`, `Intro.tex`, `Conclusions.tex`, `Appendix.tex`: Титульный лист, введение, заключение и приложения.
- `Chapter[1-3].tex`: Файлы с содержанием глав работы.
- `Bib.bib`: Файл библиографии в формате BibLaTeX.
- `fig/`: Каталог с изображениями, используемыми в отчете НИР.
- `practice/`: Каталог с программной реализацией модели на Python.
